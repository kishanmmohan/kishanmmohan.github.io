# kishanmmohan.dev

Personal portfolio website built with Nuxt 3, Vue 3, and shadcn-vue.

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm run dev
```

## Build & Deploy

```bash
# Generate static site
pnpm run generate

# Output will be in .output/public/
# Deploy the contents of .output/public/ to GitHub Pages
```

## Tech Stack

- Nuxt 4.2.1
- Vue 3
- Tailwind CSS
- shadcn-vue
- Deployed on GitHub Pages
